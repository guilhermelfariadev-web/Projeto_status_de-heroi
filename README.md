# Projeto_status_de-heroi
Sistema de gerenciamento de status de um herói em Python utilizando Programação Orientada a Objetos (POO), com evolução de nível baseada em experiência (XP).


## 📋 Funcionalidades

- Cadastro do nome do herói
- Sistema de níveis baseado em XP
- Treinamento do herói
- Aumento de Vida e Energia
- Atualização automática do nível
- Menu interativo no terminal

## 🏆 Sistema de Níveis

| XP | Nível |
|----|--------|
| 0 - 1000 | Ferro |
| 1001 - 2000 | Bronze |
| 2001 - 5000 | Prata |
| 5001 - 7000 | Ouro |
| 7001 - 8000 | Platina |
| 8001 - 9000 | Ascendente |
| 9001 - 10000 | Imortal |
| Acima de 10000 | Radiante |

- Programação Orientada a Objetos (POO)
